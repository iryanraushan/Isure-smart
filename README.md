## Setup project

- `git clone https://github.com/iryanraushan/Isure-smart`
- `npm install`
- Login to [Sign up](https://auth0.com/) create application and follow the instructions given by "auth0"
- Add your 'Domain id' and 'client id' [main.jsx](./src/main.jsx)
- `npm run dev`
